# SkyPath — Connection Flight Search 

A Full-Stack project that finds all the direct and multi-stop flight itineraries based on a real dataset of ~260 flights across 25 airports.

## How to Run

**Interface**
The application only takes the name of the airport code as input and not the name of the place for example JFK not New York. The same applies to the destination.

**Requirements:** Docker and Docker Compose must be installed.

```bash
git clone https://github.com/sharique2004/skypath.git
cd skypath
docker-compose up
